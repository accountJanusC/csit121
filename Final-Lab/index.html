Introduction to CSIT121 concepts

For the Final Lab of CSIT121, students are to showcase their previous lab content and demonstrate key concepts learned in the class. So the primary topics I'll cover today are images, navigation, tables, & what are considered to be the best practices of website creation and design per World Wide Consortium standards (or more specifically Web Content Accessibility Guidelines aka WCAG).As an overview, I'll give a short definition of the 4 topics and mention the tags involved to successfully utilise them in a website.

Overview and Final Reflection

  • Created by Tim Berners-Lee and other top-tier programmers (like HÃ¥kon Wium Lie), the World Wide Web Consortium is an organisation dedicated to making the web accessible to everyone. They have released WCAG and other standards to make sure those with disabilities or on different devices and operating systems can access the web smoothly and securely.
  • Images or graphics are stored in various formats such as PNG, JPEG, GIF, SVG, and others not covered in the scope of the class (like TIFF, PSD, BMP etc.). Images can be added on a website using HTML or CSS (and also through other languages such as Javascript and Java but those are beyond the scope of the class).
  • Navigation can refer to both the general method of moving between webpages on a website as well as a navigation bar. The 'nav' tag is used for the latter. Notable table tags are 'th' which stands for table header, 'tr' which stands for table row, and 'td' which stands for table data.
