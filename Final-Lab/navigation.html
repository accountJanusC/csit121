Explanation of Navigation elements

Navigation ties in with some WCAG concepts, such as Operability & Understandability. Having a consistent navigation (in this case, a 'nav' element) without faulty links to each webpage ensures that most people can navigate smoothly around the website. Media queries (relevant to the general idea of navigation) are also important for another WCAG principle, Robustness, since its function in screen resizing allows both mobile and desktop users to view webpages in a user-friendly way. (This W3Schools article found in Module 05 helped me understand some more practical applications for media queries without use of trickier elements like grids and floats.)
