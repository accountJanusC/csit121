Lab 04 Reflection

In creating the home page for this lab, I familiarised myself with how to add a navigation bar as well as an image with the figure and figcaption elements. I originally prepared an embedded stylesheet, which ended up being a squandered effort since I later saw that an external one was preferred. I also found out that moving the images used here to the 'Images' folder caused it to fail to load, likely due to being unable to update the file path. Hence, I needed to keep both the image and html files all in the same folder.

Lab 08 Reflection

To modify my navigation bar into one that is "sticky", I needed to familiarise myself with the W3Schools tutorial provided in the assignment description. I learned that the 2 most important aspects to create the sticky effect is setting the CSS position property to 'sticky'. The top property I also didn't understand so the MDN article on it helped me grasp how to toggle it to different locations. The W3Schools article on the overflow property also helped me understand its purpose in my stylesheet.

I did not need to use JavaScript to create the sticky navbar. In the W3Schools tutorial, I initially had no idea what the javascript:void(0) syntax did but a freeCodeCamp article cleared things up in explaining that it serves to prevent reloading and navigation to other pages.
